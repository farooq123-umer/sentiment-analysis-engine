# Sentiment Analysis Engine

A text classification project that detects whether a review is positive, negative, or neutral.

## What I used
- Python
- Scikit-learn (Naive Bayes, Logistic Regression)
- NLTK, Pandas, Matplotlib

## What I did
- Preprocessed 10,000+ reviews using tokenization and TF-IDF vectorization
- Trained and compared Naive Bayes and Logistic Regression classifiers
- Achieved around 88% accuracy on the test set
- Plotted confusion matrices and ROC curves to evaluate results

## What I learned
This was my first NLP project. I learned how text preprocessing works and how to turn raw text into something a model can understand.
